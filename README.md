# Módulo 3 - Estudo de LangChain

Este repositório contém a pasta `modulo_3`, criada com o objetivo de estudar o framework LangChain e seus principais conceitos.

## Objetivo

O conteúdo foi desenvolvido para aprender e praticar o uso de LangChain, incluindo:

- Criação de prompts e templates
- Uso de encadeamentos simples (chains)
- Uso de roteadores (router chains)
- Carregamento de dados
- Exercícios práticos sobre todo o conteúdo estudado

## Estrutura da pasta

Os arquivos principais incluem exemplos e testes de diferentes conceitos do módulo:

- `01.py` - Primeiro exemplo de uso, possivelmente com introdução e configurações iniciais.
- `02_Caching de prompts.py` - Demonstração de técnicas de cache de prompts para otimização de chamadas.
- `03_prompt_template.py` - Exemplo de criação e uso de templates de prompt.
- `04_Simple_Chains.py` - Mostra como construir encadeamentos simples (simple chains).
- `05_Router Chains.py` - Mostra como criar e usar roteadores de chains.
- `06_loaders.py` - Exemplos de carregamento de dados e integração com fontes externas.
- `base_conhecimento.csv` - Base de conhecimento em formato CSV para uso nos exemplos.
- `base_conhecimento.txt` - Base de conhecimento em formato TXT.
- `enunciado_exercicio.txt` - Enunciado do exercício realizado.
- `exercicio.py` - Código do exercício prático.
- `interface_exercicio.py` - Interface ou componente adicional do exercício.
- `requirements.txt` - Dependências necessárias para rodar os exemplos.

## Observações

Esse trabalho foi feito como parte de um estudo direcionado a LangChain e para consolidar o aprendizado através de um exercício prático que cobre todo o conteúdo estudado.
